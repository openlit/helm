## Introduction

This is the repository for OpenLIT Helm charts. This repo contains the following charts:

| Chart             | Path                               | Description                                                             |
|-------------------|------------------------------------|-------------------------------------------------------------------------|
| openlit           | `charts/openlit`                   | Primary Helm chart for deploying the OpenLIT                   |
| opentelemetry-gpu-collector | `charts/opentelemetry-gpu-collector` | Helm chart for deploying the OpenTelemetry GPU Collector for GPU monitoring |


## üíø Usage

To add the OpenLIT Chart repo, run the following command:

```bash
helm repo add openlit https://openlit.github.io/helm/
```
